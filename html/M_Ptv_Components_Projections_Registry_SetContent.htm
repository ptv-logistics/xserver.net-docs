csv
Type: SystemString
String containing the coordinate reference systems in the format id;wkt, one CRS per line.
allowReplace (Optional)
Type: SystemBoolean
Specifies if an existing coordinate reference system should be replaced, if it exists.
reinitialize (Optional)
Type: SystemBoolean
Specifies if the registry is cleared before adding the coordinate reference system specified by csv.
Remarks
Note that allowReplace and reinitialize are not mutually exclusive. allowReplace is considered per record and has an effect on duplicates included in the csv data.
