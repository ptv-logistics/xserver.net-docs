limit
Type: SystemInt32
Size limit of the LRU cache.
size (Optional)
Type: SystemFuncT, Int32
Optionally provides the function to calculate the size of an element. If the size of the LRU cache is to be restricted simply by the number of elements, the size function should return a constant value of "1". This is also the default that is used in case the size parameter is set to null. If you want to limit the LRU cache by a real size, use the size function to return the byte size of an element.
