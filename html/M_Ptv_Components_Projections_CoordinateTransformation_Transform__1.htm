Remarks

Please note that this transformation routine is restricted to object types. It cannot handle structs and can therefore not be used for sets containing Location or Point.

setLocation is called for every transformation, passing in the original object.

Examples
The following is an example for transforming some places from PTV GeoMinSek to PTV Mercator:
Copy
class Place
{
   // place location
   public Location Location;

   // place name
   public String Name;
}

...

// create some places
 Place[] places = new Place[] {
     new Place() { Location = new Location(841090, 5006420), Name = "Frankfurt" },
     new Place() { Location = new Location(1323560, 5230020), Name = "Berlin" }
 };

 // get a transformation for transforming from PTV GeoMinSek to PTV Mercator
 ICoordinateTransformation t = CoordinateTransformation.Get(
     CoordinateReferenceSystem.Mapserver.cfGEOMINSEK,
     CoordinateReferenceSystem.XServer.PTV_MERCATOR
 );

 // transform the places
 t.Transform<Place>(
     places, place => place.Location,
     (place, location) => place.Location = location
 );
